## English

Public repository for feedback on **River Navigator** (iOS) — a navigator for
paddling trips and river journeys: offline maps, GPX routes, map notes.

### 🐞 Found a problem or have an idea?

1. Open the **[Issues](../../issues)** tab
2. Click **New issue**
3. Pick a template — "🐞 Report a problem" or "💡 Suggest an improvement" (English or Russian)
4. Fill it in and click **Submit new issue**

👉 Direct link: **<https://github.com/rmsbeetle/river-navigator-public/issues/new/choose>**

Creating an issue needs a free GitHub account. If you'd rather not sign up, email
**samoilov.oleg@icloud.com**, or use TestFlight: take a screenshot and tap Share to
send feedback to the developer.

## Русский

Публичный репозиторий для обратной связи по приложению **River Navigator** (iOS) —
навигатор для сплавов и водных походов: оффлайн-карты, маршруты из GPX, точки на карте.

### 🐞 Нашли проблему или есть идея?

1. Откройте вкладку **[Issues](../../issues)**
2. Нажмите **New issue**
3. Выберите шаблон — «🐞 Сообщить о проблеме» или «💡 Предложить улучшение» (на русском или английском)
4. Заполните и нажмите **Submit new issue**

👉 Прямая ссылка: **<https://github.com/rmsbeetle/river-navigator-public/issues/new/choose>**

Для создания issue нужен бесплатный аккаунт GitHub. Если регистрироваться не хотите —
напишите на **samoilov.oleg@icloud.com**, или прямо в TestFlight: сделайте скриншот и
нажмите «Поделиться» — отзыв придёт разработчику.
